# CaterLoop Meal Planner

## Overview
CaterLoop Meal Planner is a web application designed to help users plan meals for their teams. The application allows users to select different meal options, set timing preferences, and customize their meal plans according to their needs.

## Features
- **Meal Card Selection**: Users can choose from various meal cards that represent different meal options.
- **Dynamic Timing Options**: The available timing options update dynamically based on the selected meal card.
- **Custom Time Selection**: Users can select their own meal delivery time using a clock interface.
- **Meal Name Input**: Users can set a custom name for each meal they plan.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `src` directory and open `lastform.html` in your web browser to view the application.
3. Ensure that all necessary files are in place as per the project structure.

## Usage Guidelines
- Select a meal card to view available timing options.
- Use the clock interface to set a custom delivery time.
- Enter a meal name in the provided input box to personalize your meal plan.
- Follow the prompts to complete your meal planning process.
